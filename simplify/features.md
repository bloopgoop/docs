# Features
{: .fs-9 }

## v1.0.0 Features
- Offline audio playback
- Create playlists
- Register songs to a playlist
- Edit playlists
- Edit song image
- Loop current track
- Shuffle song queue
- View your song stats
- Compatible with macOS, Windows
- Manages music files

## Future Plans
- Search song by title, album, and artist
- Import playlists
